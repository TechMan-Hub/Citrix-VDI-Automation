# Citrix VDI Automation Project

## Objective
Automate power-on and shutdown of Citrix VDI desktops for maintenance and scanning.

## What I Built
- Power On script (handles assigned vs unassigned machines)
- Power Off script (resets environment)
- Task Scheduler automation

## Key Problems Solved
- Fixed authentication issues in scheduled tasks using API profile
- Resolved delivery group matching issues
- Handled unassigned desktops with maintenance mode
- Identified why logs appeared empty during successful runs

## Result
- Automated 350+ desktops
- Eliminated manual weekend work
- Achieved full production success
